# Telas do Uber – Aventura 1: O Ataque dos Clones

Este projeto foi desenvolvido como parte da **Aventura 1** da disciplina, cujo objetivo é clonar duas telas de algum aplicativo real utilizando **React Native com TypeScript**.

O desafio consiste em criar duas telas, separar estilos, usar componentes básicos e entregar o projeto no Git com a imagem referente ao app escolhido.

---

## Sobre o Projeto

Como iniciante em React Native, optei por realizar um clone simples:

- **Tela 1:** Apenas a **logo do Uber**, criada dentro do arquivo `App2.jsx`.
- **Tela 2:** A tela principal com **opções simulando o app**, desenvolvida em `App.jsx`.

O objetivo foi compreender a estrutura do React Native, organização de componentes e separação de estilos usando `StyleSheet`.
